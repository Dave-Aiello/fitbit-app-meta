# OpenClaw Fitbit App Metadata

This repository hosts required public pages for a personal Fitbit Web API app:

- Terms of Service (`terms.html`)
- Privacy Policy (`privacy.html`)
- Landing page (`index.html`)

## GitHub Pages URLs

After enabling GitHub Pages on `main` branch (root), use:

- Website: `https://dave-aiello.github.io/fitbit-app-meta/`
- Terms: `https://dave-aiello.github.io/fitbit-app-meta/terms.html`
- Privacy: `https://dave-aiello.github.io/fitbit-app-meta/privacy.html`
